Semana 2 - Diseño de Interfaz con Flutter

## Descripción
Esta aplicación muestra un producto (“Maleta Escolar Multiuso”) con su imagen, descripción, colores disponibles y botones de interacción. Se utiliza como ejemplo de diseño de interfaz en Flutter.

El archivo main.dart contiene toda la interfaz de la aplicación. Se utiliza un Column como contenedor principal. En la parte superior, se usa un Row que contiene el icono de regreso, el título del producto y el icono de estrella para indicar favorito. Debajo, se muestra la imagen del producto utilizando Image.network. A continuación, se agrega un texto descriptivo con Text, explicando las características del producto. Después, se incluye un Row con varios Container circulares que representan los colores disponibles del producto. Finalmente, en la parte inferior, hay otro Row que contiene un ElevatedButton para añadir el producto al carrito y un Icon de corazón.
